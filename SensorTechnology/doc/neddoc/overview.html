SensorForOmnet

This project uses and extends Omnet++ and the MiXiM to get a sensor technology integration in the simulation. It contains different modules to describe the different elements a sensor node needs:

  • processor
  • different sensors
    • temperature
    • light
    • humidity
    • pressure
  • each sensor has the following modules
    • sensing unit
    • signal conditioner
    • signal converter
    • transducer
  • memory
  • from MiXiM
    • transceiver modules
    • battery support
